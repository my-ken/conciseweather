# Concise Weather
My first personal application -- Ken


##简介
  本人Android学习之路上的第一个应用，该应用参考了郭霖的《第一行代码》中的实战部分，
  综合了Activity、BroadcastReceiver、Service、网络、数据库、UI、JSON数据解析、数据存储等知识的运用。<br>
  该应用使用Android Studio进行开发。<br>
  天气数据源自[和风天气](http://www.heweather.com)。


##功能介绍


* 预置全国绝大部分市县城市数据，支持本地搜索；
* 直观显示天气数据，支持7天天气；
* 支持每8小时自动更新；
* 支持切换城市及手动更新天气功能。
